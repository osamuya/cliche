# cliche

通常Webアプリで使われる一般的な機能のひな形を作成しておくのが目的です。

- 認証機能で確認画面やメール認証などの新規登録フローの改良。
- ユニークバリデーションの際にdelflagを参照するなどのカスタムバリデーションの搭載
- usersテーブルからIDで紐付いたjoinテーブルの作成とひな形（手順）
- カスタマイズの手順書


## Set up

### Git clone
````
git clone git@github.com:osamuya/cliche.git
````
